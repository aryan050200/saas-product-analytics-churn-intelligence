# 📊 SaaS Product Analytics & Churn Intelligence Platform

An end-to-end Data Analytics project that leverages **Python, SQL, Machine Learning, Prophet Forecasting, and Power BI** to analyze customer behavior, predict churn, forecast recurring revenue, and deliver actionable business insights for a SaaS company.

---

## 🚀 Project Overview

This project simulates a real-world SaaS analytics workflow by transforming raw subscription data into business insights through data cleaning, SQL analysis, predictive modeling, revenue forecasting, and interactive dashboards.

### Business Objectives

- Predict customers at risk of churn
- Analyze subscription and revenue performance
- Forecast future Monthly Recurring Revenue (MRR)
- Evaluate customer support performance
- Build executive dashboards for decision-making

## ✨ Key Features

- Cleaned and transformed multi-table SaaS datasets using Python
- Performed advanced SQL analysis using JOINs, CTEs, Window Functions, Views, and Business Queries
- Built multiple churn prediction models and selected **XGBoost** as the best-performing model
- Forecasted the next **6 months of Monthly Recurring Revenue (MRR)** using Prophet
- Designed an interactive **5-page Power BI dashboard** for executive reporting
- Generated business insights to support customer retention and revenue growth

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/aryan050200/saas-product-analytics-churn-intelligence.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the notebooks in the following order:

- `01_eda_cleaning.ipynb`
- `02_churn_model.ipynb`
- `03_forecasting.ipynb`

4. Open the Power BI dashboard using the `.pbix` file.
